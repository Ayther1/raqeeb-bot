# 🤖 رقيب | RaqeebIQBot

بوت تيليجرام لمتابعة سوق الأسهم العراقية

---

## 📁 هيكل المشروع

```
raqeeb/
├── main.py          ← تشغيل البوت
├── config.py        ← الإعدادات
├── scraper.py       ← سحب البيانات من ISX
├── handlers.py      ← ردود البوت
├── scheduler.py     ← الجدول الزمني
├── database.py      ← قاعدة البيانات
├── messages.py      ← قوالب الرسائل
└── requirements.txt ← المكتبات
```

---

## ⚙️ طريقة التشغيل

### 1. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 2. تشغيل البوت
```bash
python main.py
```
